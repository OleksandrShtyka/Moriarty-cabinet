# 💜 Личный Кабинет Семьи MORIARTY (GTA5RP Murrieta)

Добро пожаловать в премиальный веб-кабинет семьи **Moriarty** на сервере **GTA5RP Murrieta**! Этот личный кабинет сочетает в себе потрясающий неоново-фиолетовый космический дизайн, полную интерактивность, интеграцию с облачной СУБД **Supabase** и искусственный интеллект — вашего преданного слугу по имени **«Личный раб Володя»** на базе **Google Gemini API**.

---

## 🚀 Быстрый запуск (Локальный запуск)

Проект полностью автономен и работает на чистом HTML, CSS и Vanilla JS. Установка дополнительных библиотек (`npm`/`node`) не требуется.

### Вариант 1: Через локальный веб-сервер Python (Рекомендуется)
Для корректной работы всех модулей рекомендуется запустить локальный сервер в папке проекта.
1. Откройте консоль/терминал в папке проекта.
2. Выполните команду:
   ```bash
   python -m http.server 8000
   ```
3. Откройте в браузере: [http://localhost:8000](http://localhost:8000)

### Вариант 2: Прямой запуск
Просто дважды кликните по файлу `index.html` для запуска в браузере.

По умолчанию сайт запускается в **Демо-режиме** (LocalStorage вашего браузера выступает базой данных). Вы можете сразу протестировать весь функционал:

### Данные для демо-входа:
* **Владелец семьи (OWNER)**:
  * **Email**: `owner@moriarty.fam`
  * **Пароль**: `owner`
* **Разработчик (Developer)**:
  * **Email**: `developer@moriarty.fam`
  * **Пароль**: `developer`
* **Модератор (MODERATOR)**:
  * **Email**: `moderator@moriarty.fam`
  * **Пароль**: `moderator`
* **Участник (MEMBER)**:
  * **Email**: `member@moriarty.fam`
  * **Пароль**: `member`

*(Пароль для демо-аккаунтов совпадает с префиксом их почты!)*

## 🗄️ Подключение к реальной БД Supabase

Чтобы перевести кабинет в боевой облачный режим, выполните эти шаги:

1. **Создайте проект** на [Supabase.com](https://supabase.com/).
2. Перейдите во вкладку **SQL Editor** в консоли Supabase.
3. Откройте файл `database.sql` из папки кабинета, скопируйте его содержимое и вставьте в редактор SQL Supabase.
4. Нажмите кнопку **Run** в правом нижнем углу редактора. Это автоматически создаст все таблицы (`profiles`, `warns`, `feedback`, `transactions`, `system_settings`), настроит политики безопасности (RLS) и триггер автосоздания профиля при регистрации.
5. Перейдите в настройки Supabase вашего проекта: **Project Settings** -> **API**.
6. Скопируйте **Project URL** и **Anon API Key** (public).
7. Зайдите в ваш кабинет под любым аккаунтом, перейдите в раздел **«Подключение»** (Settings) и введите скопированные ключи.
8. Нажмите **«Сохранить и подключиться»**. Кабинет мгновенно переключится на Supabase, активируя облачную синхронизацию данных и реальную авторизацию!

## 🤖 Настройка ИИ «Личный раб Володя» (Gemini API)

Володя — это верный слуга семьи Moriarty с жестким системным промптом раба, готовый отвечать на любые приказы. Для его работы:

1. Получите бесплатный или коммерческий ключ Gemini API на [Google AI Studio](https://aistudio.google.com/).
2. Скопируйте ключ (`AIzaSy...`).
3. Перейдите в кабинете в раздел **«Подключение»** и введите ключ в поле **Google Gemini API Key**.
4. Сохраните настройки.
5. Перейдите во вкладку **«Раб Володя»** и начните общаться! Он сразу же подхватит контекст раба Moriarty и будет общаться с вами на русском языке в своем неповторимом покорном стиле.

## 🛡️ Панель Администратора (Админка)

Если вы вошли под ролью **OWNER** или **Developer** (например, под демо-аккаунтами `owner` или `developer`), в левом меню появится вкладка **«Админка»**.

### Возможности админки:
1. **Управление составом**: поиск участников по имени или статическому ID (CID).
2. **Смена ролей**: повышение участников до Модератора (`MODERATOR`), Разработчика (`Developer`) или Владельца (`OWNER`).
3. **Управление выговорами (варнами)**: возможность выдать или снять варны пользователю (при получении 3 варнов статистика предупредит участника).
4. **Управление балансом**: начисление / списание средств со счета участника.
5. **Разум раба Володи**: вы можете изменять глобальный системный промпт ИИ в реальном времени! Напишите новые правила поведения в текстовом поле внизу админки, нажмите **«Обновить разум раба Володи»**, и он моментально изменит свой стиль общения для всех пользователей.

## 🎨 Технологический стек и Дизайн

* **Структура**: HTML5 (семантическая верстка, SEO-оптимизированные теги и уникальные ID).
* **Стили**: Ванильный CSS3 с использованием CSS-переменных, премиальных шрифтов Orbitron и Inter, адаптивной сетки Grid/Flexbox и эффекта стекломорфизма (Glassmorphism + `backdrop-filter: blur()`).
* **Логика**: Чистый JavaScript ES6 с асинхронными вызовами `fetch` к API Google Gemini и интеграцией с Supabase JS.
